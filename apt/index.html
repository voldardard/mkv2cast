This is the official APT repository for mkv2cast.

Installation

Add the repository to your system:

# Add the repository
echo "deb [trusted=yes] https://voldardard.github.io/mkv2cast/apt stable main" | sudo tee /etc/apt/sources.list.d/mkv2cast.list

# Update and install
sudo apt update
sudo apt install mkv2cast
Note: This repository uses [trusted=yes] as packages are not GPG signed. For production use, consider using the PyPI package: pip install mkv2cast

Alternative Installation Methods

  • PyPI: pip install mkv2cast
  • AUR (Arch Linux): yay -S mkv2cast
  • Manual: Download .deb from GitHub Releases
